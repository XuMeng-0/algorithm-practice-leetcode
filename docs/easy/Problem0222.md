# 222. 完全二叉树的节点个数

> &zwnj;**原题链接**&zwnj;: https://leetcode.cn/problems/count-complete-tree-nodes  
> &zwnj;**难度**&zwnj;: 简单

## 题目描述

给你一棵 完全二叉树 的根节点 root ，求出该树的节点个数。

完全二叉树的定义：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，
并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第 h 层（从第 0 层开始），则该层包含 1 ~ 2h 个节点。

## 解题思路

1. 步骤一: 判断根节点是否为 null，若是，则节点个数为 0
2. 步骤二：计算左子树的节点个数，记为 l
3. 步骤三：计算右子树的节点个数，记为 r
4. 步骤四：计算总节点个数， l + r + 1

左子树、右子树的节点个数计算方法和计算整棵树的方法相同。
先判断子树根节点是否为 null，若是，则子树的节点个数为 0。
然后分别计算子树的根节点的左右子树的节点个数，求和再加一，得到子树的节点个数。
