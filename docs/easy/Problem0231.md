# 231. 2 的幂

> &zwnj;**原题链接**&zwnj;: https://leetcode.cn/problems/power-of-two  
> &zwnj;**难度**&zwnj;: 简单

## 题目描述

给你一个整数 n，请你判断该整数是否是 2 的幂次方。如果是，返回 true ；否则，返回 false 。

如果存在一个整数 x 使得 n == 2x ，则认为 n 是 2 的幂次方。

## 解题思路

方案一：将给定的数字依次和 2 的幂次方比较，如果存在相同的，则是 2 的幂次方，否则不是。

方案二：将给定的数字转换为 2 进制形式，根据二进制形式中数字 1 的位数判断。如果 1 的位数大于 1 ，则不是 2 的幂次方，否则是。
