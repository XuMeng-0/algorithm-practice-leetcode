# 242. 有效的字母异位词

> &zwnj;**原题链接**&zwnj;: https://leetcode.cn/problems/valid-anagram  
> &zwnj;**难度**&zwnj;: 简单

## 题目描述

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词 。

字母异位词：通过重新排列不同单词或短语的字母而形成的单词或短语，并使用所有原字母一次。

## 解题思路

1. 步骤一：使用键值对集合 Map 记录每个字符在字符串中出现的次数，字符为键，次数为值
2. 步骤二：依次对比相同字符在两个字符串中的出现的次数是否相同，若全部相同，则是字母异位词，否则不是
