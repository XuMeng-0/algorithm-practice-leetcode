# 226. 翻转二叉树

> &zwnj;**原题链接**&zwnj;: https://leetcode.cn/problems/invert-binary-tree  
> &zwnj;**难度**&zwnj;: 简单

## 题目描述

给你一棵二叉树的根节点 root ，翻转这棵二叉树，并返回其根节点。

## 解题思路

1. 步骤一：判断根节点是否为 null，若是，则返回 null
2. 步骤二：如果左子树的根节点不是 null，则翻转左子树
3. 步骤三：如果右子树的根节点不是 null，则翻转右子树
4. 步骤四：交换根节点的左右子树

左子树、右子树的翻转方法和整棵树的翻转方法相同。
先判断子树的根节点是否为 null，若是，则返回为 null。
然后分别子树的左右子树，交换子树的左右子树，得到翻转后的子树。
