# 228. 汇总区间

> &zwnj;**原题链接**&zwnj;: https://leetcode.cn/problems/summary-ranges  
> &zwnj;**难度**&zwnj;: 简单

## 题目描述

给定一个无重复元素 的 有序 整数数组 nums ，nums 按升序排列。

区间 [a,b] 是从 a 到 b（包含）的所有整数的集合。

返回 恰好覆盖数组中所有数字 的 最小有序 区间范围列表 。
也就是说，nums 的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个区间但不属于 nums 的数字 x 。

列表中的每个区间范围 [a,b] 应该按如下格式输出：

- "a->b" ，如果 a != b
- "a" ，如果 a == b

## 解题思路

确定区间的起止位置，将区间添加到集合。重复直到区间的终止位置是数组的最后一个元素。

1. 步骤一：确定区间的起始位置，第一个起始位置为 0 ，之后的起始位置为上一个区间的终止位置的下一个位置。
2. 步骤二：确定区间的终止位置，从起始位置开始，判断下一个元素和当前元素是否连续，找到第一个不连续位置作为终止位置。
3. 步骤三：添加区间到集合中，根据起止位置是否相同，分别添加对应形式的元素。

特殊情况，空数组直接返回空集合。
